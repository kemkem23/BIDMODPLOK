1) ขนาดงาน และกริดจัดวาง

อัตราส่วน: 1:1 (สี่เหลี่ยมจัตุรัส)

ขนาดแนะนำ: 2048×2048 px (หรือทำเป็น 1080×1080 ก็ได้ แต่คงสัดส่วนเดิม)

Safe margin: เว้นขอบรอบภาพ ~4–5% ของความกว้าง (กันโดนตัดบนแพลตฟอร์ม)

แกนหลัก: ทุกอย่างจัดกึ่งกลางแนวตั้ง (center axis) แต่มีองค์ประกอบ “เอียง/สปีด” ซ้าย-ขวาเพื่อความเร็ว

2) โทนสีหลัก (Palette) + วัสดุ (Look & Feel)

ภาพนี้คือสไตล์ Racing / eSport / Speed UI โทน “ดำ-แดง-ขาว” และมีเขียวเป็นสีไฮไลต์

พื้นหลัง: ดำไล่เฉด (charcoal → black) เช่น #050505, #0B0B0B

แดงหลัก: แดงสด/แดงเข้มสำหรับเส้นสปีดและกรอบ เช่น #D10000 / #A00000

ขาว: สำหรับตัวอักษร/ไฮไลต์ความคม เช่น #FFFFFF

เทาเมทัลลิก: สำหรับขอบ/เงาโลหะ เช่น #BFC3C7, #6D6D6D

เขียวไฮไลต์: ใช้เน้นช่อง “กำลังเลือก/ดีที่สุด” เช่น #39FF14 หรือ #2DFF57 (โทนเขียวนีออน)

วัสดุภาพรวม:

แผง UI เป็น แผ่นดำด้าน (matte) + กรอบแดงนีออน + ขอบโลหะ/คมมุม

มี เงาทอด (drop shadow) และ inner shadow เพื่อให้แผงลอยจากพื้นหลัง

3) พื้นหลัง (Background) และเส้นความเร็ว (Speed Lines)

พื้นหลังทำให้รู้สึก “กำลังพุ่งไปข้างหน้า”

ทำ เส้นเฉียงแนวนอน พุ่งจากซ้ายไปขวา (หรือขวาไปซ้าย) เป็นชั้นๆ

เส้นแบ่งเป็น 3 สีหลัก: แดง / ขาว / ดำเข้ม

ลักษณะเส้น:

ยาว ปลายเรียว (tapered)

มี motion blur เบาๆ

ซ้อนหลายชั้น ความหนาไม่เท่ากัน

วางเส้นหนักด้าน มุมซ้ายบน และ มุมขวาบน เพื่อ “ครอบ” สายตาเข้าหากึ่งกลาง

กลางภาพปล่อยโล่งกว่าเล็กน้อย เพื่อให้ตัวอักษรและแผงอ่านง่าย

4) โลโก้/ตราสัญลักษณ์ด้านบน (Top Emblem)

ตำแหน่ง: กึ่งกลางด้านบน สูงประมาณ 8–12% ของภาพ
องค์ประกอบโลโก้ให้ความรู้สึก “ทีมแข่ง/สำนักช่าง”

รูปทรงโดยรวม: ตราโล่ (shield/badge) มีกรอบแดง + ขาว + เงาดำ

มี ไอคอน แนวฮาร์ดแวร์/ชิ้นส่วน (เช่นมือกำลังจับเครื่องมือ) วาดแบบการ์ตูนคมๆ (vector)

ใต้โลโก้มี ลายธงตาหมากรุก 2 ผืนไขว้กัน (เป็นสัญลักษณ์สนามแข่ง)

เพิ่มเส้นสปีดเล็กๆ รอบโลโก้ ให้เหมือนโลโก้ “พุ่งออกมา”

เอฟเฟกต์:

โลโก้มี ไฮไลต์ขาว ด้านบน และ เงาทอด ด้านล่าง

ขอบแดงมี glow จางๆ

5) ป้ายหัวข้อหลักแบบแผง (Main Title Plate)

ถัดลงมาจากโลโก้ วางเป็น “แผงใหญ่” กึ่งกลาง

ความกว้างแผง: ~60–70% ของภาพ

ความสูงแผง: ~10–12%

รูปทรง: สี่เหลี่ยมผืนผ้า มุมตัด/มุมเว้า (sci-fi panel) มีชิ้น “หูจับ/บ่ากรอบ” ซ้ายขวา

สีพื้นแผง: ดำด้าน

กรอบ: เส้นแดง 2 ชั้น (แดงเข้มด้านนอก + แดงสดด้านใน) ให้เป็นกรอบนีออน

เพิ่มเส้นไฮไลต์สีเทาเมทัลลิกบางๆ ตรงมุม/ขอบ เพื่อให้ดูเป็นชิ้นโลหะ

ข้อความหัวข้อในแผง:

ฟอนต์ไทยหนา (ดู “แข่ง/สปอร์ต”) สีขาวล้วน

ใส่ drop shadow ดำบางๆ เพื่อดันให้ลอย

6) หัวข้อรอง “สไตล์คำอังกฤษใหญ่” (Secondary Headline)

ใต้แผงหัวข้อหลัก วางคำอังกฤษใหญ่แบบ “Racing Title”

กึ่งกลาง

ตัวอักษรหนา “condensed / block” และ เอียงเล็กน้อย (italic/slant) เพื่อความเร็ว

ทำ ไล่เฉดขาว→เทา หรือขาวล้วน แล้วมี

เส้นขอบแดง (stroke) ชั้นนอก

เส้นขอบขาวบาง (stroke) ชั้นใน (ถ้าต้องการความคม)

เงาทอด ลงด้านล่าง

ระยะห่างจากแผงบน: ~2–3% ของความสูงภาพ

7) การ์ดข้อมูล 2 ฝั่ง (Left/Right Lane Cards)

ส่วนกลางภาพเป็นไฮไลต์ “สองเลน” แบ่งซ้าย-ขวาแบบสมมาตร

ตำแหน่ง: อยู่ประมาณช่วง 45–60% ของความสูงภาพ

วางการ์ด 2 ใบ:

ใบซ้ายชิดซ้ายจากกึ่งกลางเล็กน้อย

ใบขวาชิดขวาจากกึ่งกลางเล็กน้อย

เว้นช่องว่างกลาง ~6–8% ของความกว้างภาพ

โครงสร้างการ์ด (ต่อ 1 ใบ)

การ์ดประกอบด้วย 3 ชั้นสำคัญ:

(A) ป้ายเล็กด้านบนของการ์ด (Lane Tag)

เป็นแท็บเล็กสีดำกรอบแดง วางบนขอบการ์ด

ตัวอักษรสีขาว (ไทย) ขนาดเล็กกว่าหัวข้อใหญ่

(B) แถบชื่อรายการหลักของการ์ด

เป็นแถบยาวสีดำ (หรือดำไล่เทาเล็กน้อย)

ขอบแดง + ขอบขาวบาง (ให้ดูคม)

ซ้าย/ขวาของแถบมี ธงตาหมากรุก เป็นไอคอนประกอบ (เหมือนปักธง)

(C) กล่องรายละเอียดด้านล่าง (Stats Box)

เป็นสี่เหลี่ยมผืนผ้าเอียงนิดๆ (มุมเฉียงให้ความเร็ว)

พื้นกล่องดำด้าน + inner shadow

มีเส้นแบ่งคอลัมน์บางๆ สีเทา/แดงเข้ม

ฝั่งซ้ายเป็นรายการข้อความเรียงบรรทัด (เช่น 4 บรรทัด)

ฝั่งขวาเป็นตัวเลข/ค่าเวลาเรียงบรรทัด

ค่า “ที่ยังไม่พร้อม/placeholder” ใช้สีแดง เพื่อให้เด่นว่าเป็นค่ารออัปเดต

เอฟเฟกต์เพิ่มความเป็น “Dashboard”

ใต้การ์ดมีเงาทอดชัดขึ้นเล็กน้อย

มุมการ์ดมีแถบไฮไลต์โลหะสีเทาเหมือนชิ้นครอบมุม

8) ส่วนตารางอันดับด้านล่าง (Leaderboard Table)

ช่วงล่างประมาณ 25–35% ของภาพเป็นตารางอันดับเต็มความกว้าง (แต่ยังมี safe margin)

เริ่มด้วยหัวข้อใหญ่กึ่งกลาง (คำอังกฤษ) สไตล์เดียวกับหัวข้อรอง:

ตัวหนา, condensed, ไล่เฉดขาว/เทา

stroke แดง + เงาทอด

ใต้หัวข้อมีคำเล็กๆ (เช่นหมวด/ตัวกรอง) สีขาวหรือเทาอ่อน

โครงสร้างตาราง

ตารางเป็นแผงยาว กว้าง ~85–90% ของภาพ

ใช้เส้นกรอบแดงเป็นหลัก และพื้นตารางดำ

แถวหัวตาราง (Header Row):

พื้นดำเข้ม

เส้นแบ่งคอลัมน์สีแดง

ชื่อคอลัมน์สีขาว ยกเว้น “คอลัมน์ที่ถูกเลือก” เป็นพื้น เขียวนีออน

แถวข้อมูล (Rows):

พื้นดำด้าน

เส้นคั่นแถวสีแดง

ช่องเวลา/ค่าที่ “เด่น” ทำพื้นเขียว หรือเขียวเป็นแท็บไฮไลต์

บางค่าทำพื้นขาว/เทาเพื่อบาลานซ์ (แต่ภาพนี้เน้นเขียวเป็นหลัก)

สัดส่วนคอลัมน์ (ไอเดียทำตามภาพ)

คอลัมน์ซ้ายสุดแคบ 2–3 ช่อง (เช่นลำดับ/รุ่น)

คอลัมน์ชื่อทีม/ชื่อรายการ “กว้างสุด” ~35–45% ของตาราง

คอลัมน์ตัวเลขเวลา 4 ช่องด้านขวา (ขนาดเท่าๆกัน)

9) ฟอนต์และสไตล์ตัวอักษร (Typography Guide)

เลือก “ฟอนต์ทรงแข่ง” ที่ หนา, แคบ, อ่านง่าย, คม

หัวข้ออังกฤษใหญ่

แนวทางฟอนต์: condensed sans, impact-like, sporty

ตัวเลือกใกล้เคียง: Bebas Neue / Anton / Teko / Agency FB (หรือฟอนต์สปอร์ตที่คล้ายกัน)

สไตล์:

ตัวหนามาก

เอียงเล็กน้อย (สำหรับคำที่สื่อความเร็ว)

มี stroke 2 ชั้น (ขาวบาง + แดงหนากว่า) และ drop shadow

ข้อความไทย

แนวทางฟอนต์: Sans ไทยที่ลำตัวแน่น น้ำหนักหนา

ตัวเลือกใกล้เคียง: Kanit / Prompt / Bai Jamjuree (น้ำหนัก ExtraBold/Bold)

สไตล์:

สีขาว

เงาดำบางๆ

บรรทัดสั้น กระชับ เพื่อความเป็น UI

ตัวเลขเวลา/ค่า

ใช้ฟอนต์เดียวกับอังกฤษหรือ sans ที่อ่านตัวเลขชัด

น้ำหนัก SemiBold–Bold

สีขาวบนพื้นดำ / สีดำบนพื้นเขียว (เพื่อคอนทราสต์)

10) รายละเอียดกราฟิกที่ทำให้ “เหมือนภาพนี้”

เช็กลิสต์เอฟเฟกต์/ดีเทลที่สำคัญ:

Neon Red Outline: เส้นกรอบแดงที่สว่างกว่าพื้นเล็กน้อย + glow เบาๆ

Metallic Edge: มีไฮไลต์เทา/ขาวตามมุมกรอบ ให้เหมือนวัสดุโลหะ

Drop Shadow ชัด: ทุกแผงมีเงาลงด้านล่าง/เฉียงเล็กน้อย ทำให้ลอยจากพื้นหลัง

Checkered Flags: ไอคอนธงตาหมากรุกเล็กๆ ซ้ายขวาในส่วนการ์ด (ช่วยล็อกธีมสนามแข่ง)

Motion Streaks: เส้นความเร็วทั้งพื้นหลังและบางจุดบนแผง เพื่อให้ภาพ “วิ่ง”

Green Active Column: ในตาราง ใช้เขียวเพื่อสื่อ “คอลัมน์กำลังเลือก/ผลเด่น”

11) สัดส่วนรวม (เพื่อทำให้ใกล้เคียงมาก)

ถ้าคิดเป็นความสูงทั้งภาพ 100% (จากบนลงล่าง):

0–12% : โลโก้/ตราสัญลักษณ์

12–28% : แผงหัวข้อหลัก (ดำกรอบแดง)

28–38% : หัวข้อรองตัวใหญ่ (อังกฤษ)

38–60% : การ์ด 2 เลน (ซ้าย/ขวา)

60–72% : หัวข้อส่วนตาราง (อังกฤษ) + คำเล็ก

72–98% : ตารางอันดับ

มุมล่างขวาเว้นพื้นที่โล่งเล็กน้อยให้มี “ประกาย/ไอคอนตกแต่ง”
